## After Pushing: Deploy Your Site

Once files are on GitHub, deploy for free:

### **GitHub Pages (Easiest)**
1. Go to https://github.com/HS23-hub/Web-Dev
2. Settings → Pages
3. Source: `main` branch
4. Save → Site goes live at: `https://HS23-hub.github.io/Web-Dev`

### **Vercel (Recommended)**
1. Go to https://vercel.com
2. Click "New Project"
3. Select your GitHub repo
4. Click "Deploy"
5. Get a live URL instantly

### **Netlify**
1. Go to https://netlify.com
2. Click "New site from Git"
3. Select your GitHub repo
4. Auto-deploys every push
